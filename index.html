Classic Crispy Chicken Salad

American Classic Crispy Chicken Salad

Ingrédients :

  • Poulet croustillant américain
  • Laitue iceberg
  • Tomates cerises
  • Cheddar râpé
  • Croûtons
  • Sauce Ranch

Mesures :

  1. Mélanger laitue + tomates + cheddar.
  2. Ajouter le poulet croustillant.
  3. Ajouter les croûtons.
  4. Finir avec la sauce Ranch.

Italian Classic Crispy Chicken Salad

Ingrédients :

  • Poulet croustillant italien
  • Roquette
  • Tomates séchées
  • Mozzarella
  • Basilic
  • Sauce balsamique

Mesures :

  1. Mélanger roquette + mozzarella + tomates séchées.
  2. Ajouter le poulet croustillant.
  3. Ajouter du basilic.
  4. Verser la sauce balsamique.
